# Create New Command

Create a new command following command-based programming best practices:

1. Create command class (extending Command or using functional commands)
2. Add required subsystem dependencies
3. Implement initialize(), execute(), end(), and isFinished()
4. Add command to RobotContainer with appropriate triggers
5. Consider if this should be a default command or button-triggered

Ask me for:
- Command name and purpose
- Which subsystem(s) it requires
- Triggering condition (button, auto, default, etc.)
- End conditions
